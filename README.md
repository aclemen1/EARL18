# EARL 2018 London

## Build RStudio docker image with required Linux packages installed

`make build`

## Start RStudio server on port 8787 (password 'go4it')

`make start`

## Stop RStudio server

`make stop`

## Restart RStudio server

`make restart`

## Execute interactive BASH shell in running container

`make exec`

## Clean-up

`make clean`
